# 디자인 시스템 문서

Daily Planner 웹 앱의 시각적 디자인 시스템을 정의한 문서입니다.

---

## 1. 테마 시스템

### 개요
다크 테마를 기본(`:root`)으로 사용하며, 라이트 테마는 `[data-theme="light"]` 속성으로 전환합니다.

### 테마 전환 메커니즘
- **저장소**: `localStorage` 키 `planner_theme` (`'dark'` | `'light'`)
- **초기화**: 페이지 로드 시 IIFE에서 저장된 테마를 읽어 `document.documentElement`에 `data-theme` 속성 설정
- **전환 함수**: `toggleTheme()` - 현재 테마의 반대 값으로 전환
- **라이트 전환**: `document.documentElement.setAttribute('data-theme', 'light')`
- **다크 전환**: `document.documentElement.removeAttribute('data-theme')`
- **버튼 텍스트**: 다크일 때 "☀️ 밝은 테마로 변경", 라이트일 때 "🌙 어두운 테마로 변경"
- **서버 비저장**: 테마 설정은 기기별로 관리되며 서버에 동기화하지 않음

## 3. 섹션 색상 팔레트

섹션 모달에서 선택 가능한 9개 프리셋 색상 + 커스텀 색상 선택기:

| 인덱스 | 색상 코드 | 색상 이름 (참고) |
|--------|-----------|-----------------|
| 0 | `#B026FF` | 보라 (Electric Purple) |
| 1 | `#FF44CC` | 핫핑크 (Hot Pink) |
| 2 | `#FF3355` | 빨강 (Rose Red) |
| 3 | `#FF6B2B` | 주황 (Orange) |
| 4 | `#FFE633` | 노랑 (Yellow) |
| 5 | `#39FF14` | 연두 (Neon Green) |
| 6 | `#00E5FF` | 시안 (Cyan) |
| 7 | `#4D6DFF` | 파랑 (Blue) |
| 8 | `#1B0CFF` | 남색 (Indigo) |
| - | 커스텀 | `<input type="color">` 기반 자유 선택 (✎ 아이콘) |

### 색상 자동 할당
새 섹션 생성 시 `SECTION_COLORS[data.sections.length % SECTION_COLORS.length]` 공식으로 순환 배정됩니다.

### 색상 표현 방식
- **섹션 제목**: 해당 색상으로 텍스트 색 지정
- **반복 배지**: 텍스트 색 + 배경 `{color}1F` (12% 투명도)
- **캘린더 도트**: `.cal-dot` (4x4px 원형 인디케이터)
- **체크리스트 컬러 도트**: `.section-color-dot` (8x8px 원형)

## 4. 타이포그래피

### 폰트 패밀리
```css
font-family: 'Noto Sans KR', sans-serif;
```
Google Fonts CDN에서 로드: `wght@300;400;500;700`

| 굵기 | CSS `font-weight` | 사용 위치 |

| Light | `300` | (로드만 됨, 직접 사용 없음) |

| 크기 | 사용 위치 |
|------|----------|
| `24px` | 메인 제목 (`.header-title`) |
| `22px` | 통계 값 (`.stat-val`) |
| `20px` | 캘린더 월 라벨, 날짜 네비게이션 제목 |

| `14px` | 진행률 라벨, 캘린더 날짜, 식단 입력, 설정 라벨, 데이터 관리 버튼 |
| `13px` | 헤더 날짜, 섹션 타이틀, 체크리스트 식단, 미니캘린더 헤더, 몸무게 차이 |

| `8px` | 캘린더 식단 별 아이콘 |

## 5. 컴포넌트 패턴

### Border Radius

| 값 | 적용 대상 |
|----|----------|
| `50%` (원형) | 체크 버튼, 섹션 컬러 도트, 색상 선택 옵션, 캘린더 도트 |
| `18px` | 모달 컨테이너 |
| `14px` | 미니캘린더, 메모 패널, 식단 패널, 반복 패널 |
| `12px` | 카드, 아코디언 섹션, 체크리스트 항목, 통계 카드, 트래커 테이블 |
| `10px` | 버튼 (primary/secondary), 입력 필드, 캘린더 날짜, 범위 선택 모달 버튼 |
| `8px` | 미니캘린더 날짜, 아코디언 항목, 식단 입력/추가 버튼, 반복 옵션 |
| `6px` | 반복 배지, 미니캘린더 네비게이션, 항목 액션 버튼 |
| `3px` | 프로그레스 바, 스크롤바, 메모+식단 겸용 인디케이터 |

### Transition

| 대상 | 설정 |
|------|------|
| 기본 전환 | `all 0.2s` (대부분의 호버/클릭 효과) |
| 미니캘린더 날짜 | `all 0.15s` (더 빠른 반응) |
| 프로그레스 바 채움 | `width 0.4s ease` |
| 저장 완료 메시지 | `opacity 0.3s` |

### 호버 효과 (Hover)

- **항목 (`.item`)**: 테두리 색 `rgba(124,92,252,0.25)`로 변경
- **항목 액션 버튼**: `opacity: 0` → `opacity: 1` (호버 시 노출)
- **삭제 버튼**: 호버 시 `color: var(--danger)`
- **캘린더 날짜**: 배경 `var(--bg3)`
- **색상 선택 옵션**: `transform: scale(1.15)`
- **추가 섹션 버튼**: 테두리/텍스트 `var(--accent)`, 배경 `rgba(124,92,252,0.05)`

### 모달

- **배경**: `rgba(0,0,0,0.6)` + `backdrop-filter: blur(4px)`
- **z-index**: `200` (네비게이션 `100`보다 위)
- **크기**: `width: 90%`, `max-width: 360px`
- **닫기**: 배경 클릭으로 닫기, 취소 버튼, Enter 키 저장

### 드래그 앤 드롭

- **드래그 중 원본**: `opacity: 0.25` (`.dragging`)
- **드롭 대상 위치**: `box-shadow: 0 -2px 0 0 var(--accent)` (`.drag-over`)
- **드래그 고스트**: `border-radius: 12px`, `box-shadow: 0 8px 24px rgba(0,0,0,0.4)`
- **드래그 핸들**: `cursor: grab` (활성: `cursor: grabbing`)
- **터치 지원**: `touch-action: none`, `user-select: none`

### 완료 항목

- **체크리스트 항목 (`.item.done`)**: `opacity: 0.5`
- **라벨**: `text-decoration: line-through`, `color: var(--text3)`
- **체크 버튼**: `background: var(--accent)`, `border-color: var(--accent)`
- **종료된 섹션 (`.acc-section.ended`)**: `opacity: 0.5`

### 600px 최대 너비

```css
.page { max-width: 600px; margin: 0 auto; padding: 20px; }
```
모든 페이지 컨텐츠는 600px 최대 너비로 중앙 정렬되어 모바일-우선 레이아웃을 유지합니다.

### 버튼 스타일

| 클래스 | 스타일 | 용도 |
|--------|--------|------|
| `.btn-primary` | `background: var(--accent)`, 흰색 텍스트, 700 굵기 | 모달 저장 버튼 |
| `.btn-secondary` | 투명 배경, `var(--border)` 테두리, `var(--text2)` 텍스트 | 모달 취소 버튼 |
| `.memo-save` | `background: var(--accent)`, 흰색, 700 굵기 | 메모/몸무게 저장 |
| `.cal-nav` | `var(--card)` 배경, `var(--border)` 테두리 | 캘린더/날짜 네비게이션 |
| `.data-btn` | `var(--card)` 배경, `var(--border)` 테두리 | 데이터 내보내기/가져오기 |
| `.theme-toggle-btn` | `var(--card)` 배경, 전체 너비 | 테마 전환 |
| `.add-section-btn` | 투명, 점선 테두리 (`dashed`) | 섹션 추가 |

## 6. UI 레이아웃

### 3-탭 스티키 네비게이션

```css
.nav {
  position: sticky;
  top: 0;
  z-index: 100;
  background: var(--bg2);
  border-bottom: 1px solid var(--border);
}
```

- 3개 탭 버튼이 `flex: 1`로 균등 분배
- 활성 탭: `color: var(--accent)`, 하단 `border-bottom: 2px solid var(--accent)`
- 우측에 로그인 유저 아이디 표시 (`flex-shrink: 0`)

### 탭 구성

| 순서 | 탭 이름 | 이모지 | 페이지 ID |
|------|---------|--------|-----------|
| 1 | 체크리스트 | ✅ | `page-checklist` |
| 2 | 캘린더 | 📅 | `page-calendar` |
| 3 | 일일 기록 | 📊 | `page-tracker` |

### 페이지 컨테이너

- 기본 `display: none`, 활성 시 `display: block`
- `padding: 20px`, `max-width: 600px`, `margin: 0 auto`

### 모달 z-index 체계

| z-index | 요소 |
|---------|------|
| `200` | 모달 배경 (`.modal-bg`) |
| `100` | 스티키 네비게이션 (`.nav`) |

### 통계 행 (Stats Row)

```css
.stats-row {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 8px;
}
```

3개 통계 카드 (완료/남은 항목/달성률)가 균등 3열 그리드로 배치됩니다.

각 카드 색상:
- 완료: `var(--accent)` (보라)
- 남은 항목: `var(--accent3)` (주황)
- 달성률: `var(--accent2)` (시안)

### 캘린더 그리드

```css
.cal-grid {
  display: grid;
  grid-template-columns: repeat(7, 1fr);
  gap: 4px;
}
```

- 7열 그리드 (일~토)
- 각 날짜 셀: `aspect-ratio: 1` (정사각형)
- **오늘**: `color: var(--accent)`, 700 굵기, `border-color: var(--accent)`
- **선택됨**: `background: rgba(124,92,252,0.15)`, `border-color: var(--accent)`
- **다른 달**: `opacity: 0.4`

### 미니 캘린더 (체크리스트 탭)

```css
.mini-cal-grid {
  display: grid;
  grid-template-columns: repeat(7, 1fr);
  gap: 2px;
}
```

- 토글 방식으로 표시/숨김 (`display: none` ↔ 표시)
- 동일한 7열 그리드, 더 작은 간격 (`gap: 2px`)

### 캘린더 날짜 인디케이터

| 상태 | 인디케이터 |
|------|-----------|
| 메모 있음 | `var(--accent2)` 5px 원형 도트 |
| 식단 있음 | `var(--accent3)` 5px 원형 도트 |
| 메모+식단 | `linear-gradient(90deg, var(--accent2), var(--accent3))` 12x5px 그라데이션 바 |
| 섹션 도트 | 각 섹션 색상의 4px 원형 도트 (`.cal-dots` 컨테이너) |
| 식단 별 | `var(--accent3)` 8px 별 (좌상단) |

### 아코디언 (섹션 목록)

- **섹션 패널 (`.acc-section`)**: `border-radius: 12px`, `margin-bottom: 8px`
- **헤더 (`.acc-header`)**: 클릭으로 펼치기/접기, 호버 시 `background: var(--bg3)`
- **구성**: 드래그 핸들 → 화살표 → 컬러 도트 → 제목 → 반복 배지 → 날짜 범위 → 개수 → 편집 → 삭제
- **그룹 라벨**: 활성/종료된 섹션 그룹 구분 (`acc-group-toggle`)
- **종료된 섹션**: `opacity: 0.5`

## 7. 한국어 UI 규칙

### 날짜 형식

| 위치 | 형식 | 예시 |
|------|------|------|
| 체크리스트 날짜 | `YYYY년 M월 D일 (요일)` | `2026년 4월 8일 (수)` |
| 캘린더 월 라벨 | `YYYY년 M월` | `2026년 4월` |
| 날짜 키 (데이터) | `YYYY-M-D` (zero-padding 없음) | `2026-4-8` |
| 날짜 입력 (HTML) | ISO 형식 `YYYY-MM-DD` | `2026-04-08` |

### 요일 이름

```javascript
const dayNames = ['일', '월', '화', '수', '목', '금', '토'];
```

표시 형식: `(일)`, `(월)`, ... 또는 `일요일`, `월요일`, ...

### 반복 주기 라벨

| 키 | 한국어 라벨 | 추가 표시 |
|----|------------|----------|
| `daily` | 매일 | - |
| `weekdays` | 주중 | - |
| `weekends` | 주말 | - |
| `weekly` | 매주 | `매주 {요일}요일` (예: `매주 월요일`) |
| `biweekly` | 격주 | `격주 {요일}요일` (예: `격주 수요일`) |
| `everyOtherDay` | 격일 | - |
| `monthly` | 매월 | `매월 {일}일` (예: `매월 15일`) |

### 식단 타입 라벨

| 키 | 한국어 라벨 | 이모지 |

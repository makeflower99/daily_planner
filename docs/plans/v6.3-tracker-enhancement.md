# v6.3 일일 기록 탭 기능 강화 계획

**상태:** 완료
**날짜:** 2026-04-08

## 목표
일일 기록(tracker) 탭의 기능을 강화하여 메모 확인, 습관 성장 시각화를 추가하고, 캘린더 탭의 항목 추가 UX 문제를 개선한다.

## 변경 사항

### 1. 일일 기록 탭 메모 표시
- `renderTracker()`에서 해당 날짜의 메모를 읽기 전용으로 표시
- `data.memos[trackerDate]` 접근, `escapeHtml()` 적용
- 통계 카드 아래, 체크리스트 테이블 위에 배치

### 2. 습관 변화 추적 (섹션 진화 시각화)
- `parseDateKey()` 유틸 함수 추가 (data.js)
- `computeSectionEvolution()` — 각 섹션의 시작 항목 수 vs 현재 항목 수 계산 (addedDate 기반)
- `renderEvolutionHTML()` — CSS 바 차트 + 격려 메시지 + 달성률 프로그레스 바
- 변화가 있는 섹션만 표시

### 3. 인라인 항목 추가 커서 복원 개선
- `scopeResolve()`에서 `document.activeElement.blur()` 추가 — scope 모달 포커스 해제
- `addItemInline()` 포커스 복원을 `requestAnimationFrame` + `setTimeout(50)` 조합으로 변경
- 섹션 ID 시도 순서를 `[targetSecId, secId]`로 변경 (새 섹션 우선)

## 한계점 및 향후 개선
- **삭제 이력 미추적**: addedDate만 존재하므로 삭제된 항목은 성장 그래프에 반영 불가
  - 개선안: deletedDate 필드 추가하여 삭제 시점 기록, 감소 추이도 시각화
- **항목 이름 변경 미추적**: 라벨 변경 시 별도 이벤트로 기록되지 않음
  - 개선안: 항목 변경 로그 배열 추가 (changeLog)
- **메모 읽기 전용**: 일일 기록 탭에서 편집하려면 캘린더 탭으로 이동 필요
  - 개선안: 인라인 편집 모드 (클릭 시 textarea 전환)

== 서빙 ==
처음 한 번 (npm install)
npm start
tunnel.bat
== 아이디 비번 생성 ==
node create-user.js 아이디 비밀번호

# 1. .env 파일 준비
cp .env.example .env
# .env 파일 열어서 SESSION_SECRET에 랜덤 값 입력

# 2. 유저 생성
docker compose run --rm planner node create-user.js <아이디> <비밀번호>

# 3. 서버 시작
docker compose up

docker compose up --build
docker compose down

== 알려진 한계점 (향후 개선) ==
- 습관 변화 추적: 항목 삭제 이력 미추적 — addedDate만 존재하므로 삭제된 항목은 성장 그래프에 반영 불가
  → 개선안: deletedDate 필드 추가하여 항목 삭제 시점 기록, 감소 추이도 시각화
- 습관 변화 추적: 항목 이름 변경 이력 미추적 — 같은 항목의 라벨 변경 시 별도 이벤트로 기록되지 않음
  → 개선안: 항목 변경 로그 배열 추가 (changeLog)
- 메모 표시: 일일 기록 탭에서 읽기 전용만 지원 — 편집하려면 캘린더 탭으로 이동 필요
  → 개선안: 인라인 편집 모드 추가 (클릭 시 textarea 전환)
